Fibonacci sor feladat

A program magyarázata: A program paraméterezéssel kap egy számot, ezt egy lista hosszának veszi. A program ellenőrzött bekéréssel bekér a usertől egy pozitív egész számot, és attól a számtól indulva felír egy Fibonacci számsor szerű listát, melynek hossza ugybár a paraméter. Tehát a konzolon megjelenő számsor első számjegye a beírt szám, és mivel az elv az, hogy mindig az előzőt kell hozzáadni, a következő is az lesz. Az eredeti Fibonacci sor első öt számjegye példának: 1, 1, 2, 3, 5.
